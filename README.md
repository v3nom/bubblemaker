# BubbleMaker

A Fusion 360 plugin to generate bubble-like textures on surfaces.

## Features

- Generates a bubbly/cloud-like texture on a selected surface/body.
- Customizable parameters for bubble size, depth, and distribution.

## Installation

1.  Open Fusion 360.
2.  Go to **Utilities** > **Add-Ins** > **Scripts and Add-Ins**.
3.  Click on the **Add-Ins** tab.
4.  Click the **+** button to add a new add-in.
5.  Navigate to the location of the extracted `BubbleMaker` folder and select it.
6.  Ensure "Run on Startup" is checked if you want it to be available immediately.
7.  Click **OK**.

## Usage

1.  Select a surface/body in your Fusion 360 design.
2.  Go to **Utilities** > **Add-Ins** > **Scripts and Add-Ins**.
3.  Find **BubbleMaker** in the list and click **Run**.
4.  Adjust the parameters in the dialog box:
    -   **Bubble Size**: Controls the diameter of the bubbles.
    -   **Bubble Depth**: Controls the height/depth of the bubbles.
    -   **Bubble Density**: Controls how many bubbles are generated.
5.  Click **OK** to generate the texture.
